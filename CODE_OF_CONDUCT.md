# Code of Conduct

## TL;DR

Be respectful. Be constructive. Don't be a jerk.

We follow the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct.html) v2.1. If you've contributed to open source before, you know the drill.

## Standards

**Do:**
- Be kind and patient
- Welcome newcomers
- Give constructive feedback
- Own your mistakes
- Focus on what's best for the project

**Don't:**
- Harass, insult, or attack people
- Make sexualized comments
- Dox or threaten anyone
- Be exclusionary or discriminatory
- Spam or troll

## Enforcement

Violations can be reported to the maintainers via GitHub (DM or issue) or email. All reports are confidential.

Consequences range from a warning to a permanent ban, depending on severity and pattern.

## Scope

This applies in all project spaces: GitHub (issues, PRs, discussions), any future Discord/Slack, and when representing the project publicly.

**Questions?** Ask a maintainer.
